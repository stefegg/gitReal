Git Stash

Home

Git stash is a way to store teh changes that you've made on a branch without having to commit them or add them to a project that is being worked on. It's prefect for when you're in the middle of something and you want to switch to doing something else.

Hint:

If you see the following output when changing branches, you might want to consider using git stash. "Please commit your changes or stash them before you switch branches."

To use stash, simply run:

git stash

You can view your stashed changes by running git stash list . This will show you a list of changes you have stored in the stash.

You can apply any given stash to whichever part of your project you would like by running the following command:
git stash apply stash@{#} . # = the number that corresponds to the stash that you want to apply.
